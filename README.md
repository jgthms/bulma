# [Bulma](https://bulma.io)

Bulma is a **modern CSS framework** based on [Flexbox](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout/Using_CSS_flexible_boxes).

## project Bulma with CSS grid

``` html
<!-- with grid -->
<div class="grid">
  <div>First</div>
  <div>Second</div>
  <div>Third</div>
  <div>Fourth</div>
</div>

<!-- with flexbox -->
<div class="columns">
  <div class="column">
    First column
  </div>
  <div class="column">
    Second column
  </div>
  <div class="column">
    Third column
  </div>
  <div class="column">
    Fourth column
  </div>
</div>
```
