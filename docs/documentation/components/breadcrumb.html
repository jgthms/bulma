Breadcrumb

A simple breadcrumb component to improve your navigation experience

The breadcrumb component consists of two elements:

  • breadcrumb: the main container
    • breadcrumb-item: a repeatable list item

The breadcrumb dividers are created automatically in the ::before pseudo-element's content of breadcrumb-item elements

You can inform the current page using the is-active modifier in a breadcrumb-item element. It will disable the navigation of this item.
