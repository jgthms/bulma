Dropdown

A classic toggleable dropdown containing a list of links

The structure of a dropdown is the following

  • dropdown: main container
    • is-dropdown: the button under which the dropdown-content will appear (automatically adds a caret after button label)
    • dropdown-list: the list of links in your dropdown
      • dropdown-item: a dropdown link
      • dropdown-heading: a header to label sections of actions
      • divider: a divider to separate items

No JavaScript
Bulma does not include any JavaScript interaction. You will have to implement the class toggle yourself
