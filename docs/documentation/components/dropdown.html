Dropdown

A classic toggleable dropdown containing a list of links

The structure of a dropdown is the following

  • dropdown: main container
    • is-dropdown: the button under which the dropdown-content will appear
    • dropdown-list: the list of links of your dropdown
      • dropdown-heading: a header to label sections of actions
      • dropdown-item: a dropdown item
      • divider: a divider to separate items
