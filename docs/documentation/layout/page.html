Bulma offers a handful of helper classes for determining your basic page layout.

{% include elements/anchor.html name="Sticky bottom footer" %}

The page and main classes can be used to ensure that your footer remains "stuck" to the bottom of the page, regardless of how much vertical space is taken up by other elements on the page.

Apply the page class to an HTML element that encompasses the entire page, including the footer; apply the main class to an element that encompases everything but the footer.
