Modifiers

Add borders to all the cells.

table is-bordered
One Two
Three Four

Add stripes to the table.

table is-striped
One Two
Three Four
Five Six
Seven Eight

Make the cells narrower.

table is-narrow
One Two
Three Four
Five Six
Seven Eight

You can combine all three modifiers.

table is-bordered is-striped is-narrow
One Two
Three Four
Five Six
