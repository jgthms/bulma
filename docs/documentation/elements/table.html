Tables

The inevitable HTML table, with special case cells

You simply need to attach a single .table CSS class on a <table> with the following structure:

  • table
    • thead
    • tfoot
    • tbody
      • tr
        • th
        • td

New! 0.4.2
You can set a table row as selected by appending the is-selected modifier on a <tr>

Modifiers

Add borders to all the cells.

table is-bordered
One Two
Three Four

Add stripes to the table.

table is-striped
One Two
Three Four
Five Six
Seven Eight
Nine Ten
Eleven Twelve

Make the cells narrower.

table is-narrow
One Two
Three Four
Five Six
Seven Eight
Nine Ten
Eleven Twelve

You can combine all three modifiers.

table is-bordered is-striped is-narrow
One Two
Three Four
Five Six
Seven Eight
Nine Ten
Eleven Twelve
